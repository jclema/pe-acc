# Datos crudos Peru

Coloca aqui los archivos originales descargados desde las fuentes publicas de Peru.

Estructura:

- `sunat_ruc/`: CSV original del padron RUC de SUNAT.
- `osce_sanctions/`: archivos originales de sanciones de OSCE/OECE.
- `seace_conosce/`: archivos originales de procesos, adjudicaciones y contrataciones.

Regla:

- No editar manualmente estos archivos.
- Mantener el nombre original o anotar la fecha de corte en el nombre.
