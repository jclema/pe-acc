# Datos normalizados Peru

Aqui se guardan las salidas limpias y estandarizadas de los pipelines ETL para Peru.

Estructura:

- `sunat_ruc/`: proveedores normalizados.
- `osce_sanctions/`: sanciones normalizadas.
- `seace_conosce/`: procesos, adjudicaciones y entidades normalizados.

Regla:

- Estos archivos son derivados de `data/raw/pe/...`.
- Deben ser aptos para carga a Neo4j y para validaciones intermedias.
